Introducción

El firewall constituye uno de los mecanismos fundamentales para la protección de redes y sistemas. Su función principal es controlar el tráfico entrante y saliente mediante la aplicación de políticas de seguridad previamente definidas, permitiendo únicamente las comunicaciones autorizadas y bloqueando aquellas que representen un riesgo.

En sistemas basados en Linux, una de las herramientas más utilizadas para implementar un firewall es iptables, un sistema de filtrado de paquetes que opera a nivel del kernel mediante el framework Netfilter. Iptables permite definir reglas que inspeccionan paquetes según criterios como dirección IP de origen o destino, puertos, protocolos (TCP, UDP, ICMP) y estado de la conexión.

Instrucciones

Reglas iptables

# Regla
1 iptables -A INPUT -j DROP
2 iptables -A INPUT -p tcp -m state --state ESTABLISHED -j ACCEPT
3 iptables -A OUTPUT -p tcp --sport 53 -j ACCEPT
4 iptables -A INPUT -p tcp -s 192.1.2.10 -j ACCEPT
5 iptables -A OUTPUT -p tcp -s 192.1.2.10 -j ACCEPT
6 iptables -A INPUT -p tcp --dport 80 -d 192.1.2.11 -j ACCEPT
7 iptables -A INPUT -p tcp -j ACCEPT
